Keyboard Shortcuts
------------------



### Actions

shortcuts | description
---|---
`N`  | New tweet
`L`  | Like tweet
`R`  | Reply to tweet
`T`  | Retweet
`M`  | Direct message
`U`  | Mute account
`B`  | Block account
`Enter`  | Open tweet details
`O`  | Expand photos
`/`  | Search
`Ctrl` `Enter`  | Send tweet
{.shortcuts}


### Navigation

shortcuts | description
---|---
`?`  | Full keyboard menu
`J`  | Next tweet
`K`  | Previous tweet
`Space`  | Page down
`.`  | Load new tweets
{.shortcuts}


### Timelines

shortcuts | description
---|---
`G` `N`  | Home timeline
`G` `O`  | Moments
`G` `N`  | Notifications tab
`G` `R`  | Mentions
`G` `P`  | Profile
`G` `L`  | Likes tab
`G` `I`  | Lists tab
`G` `M`  | Direct messages
`G` `S`  | Settings and privacy
`G` `U`  | Go to someone's profile
{.shortcuts}
